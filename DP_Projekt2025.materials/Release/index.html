A work of interactive fiction.

DP Projekt 2025 was created with Inform and has IFID D8FE1B9E-B7F6-432A-B713-4E064A1D68DE. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
